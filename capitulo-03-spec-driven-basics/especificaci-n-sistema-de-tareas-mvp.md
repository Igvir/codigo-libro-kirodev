# Especificación: Sistema de Tareas - MVP

## Objetivo
Validar concepto básico de gestión de tareas con funcionalidad mínima viable.

## Funcionalidades (Solo lo esencial)
- Crear tarea con título únicamente
- Marcar como completada/pendiente
- Listar todas las tareas
- Eliminar tareas

## Stack Técnico
- React 18 + TypeScript
- localStorage para persistencia
- CSS modules para estilos

## Criterios de Aceptación
- [ ] Usuario puede agregar tarea con título
- [ ] Usuario puede marcar/desmarcar como completada
- [ ] Lista muestra todas las tareas
- [ ] Usuario puede eliminar tareas
