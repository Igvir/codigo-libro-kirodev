## Development Hooks

Este proyecto usa Kiro Agent Hooks para automatizar calidad de código:

### Hooks Activos:

- **auto-test** (On File Save): Genera tests automáticamente para nuevo código
- **security-scan** (On File Save): Escanea vulnerabilidades y secretos
- **commit-message** (Manual): Genera commit messages descriptivos
- **i18n-sync** (On File Save): Sincroniza traducciones entre idiomas
