# TODO App Básica

Backend API REST:
- GET /todos - listar tareas
- POST /todos - crear tarea
- PUT /todos/:id - actualizar tarea
- DELETE /todos/:id - eliminar tarea

Modelo: id, title, completed, created_at

Frontend React:
- Lista de tareas
- Formulario para agregar
- Checkbox para marcar completadas
- Botón eliminar

Stack: Node.js + Express, React, JSON file storage (no DB)
