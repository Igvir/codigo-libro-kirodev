# Mi Steering Global

## Estilo de Código Personal
- Prefiero const sobre let cuando sea posible
- Single quotes para strings
- Trailing commas en objetos/arrays
- 2 espacios de indentación

## Comentarios
- Comenta el "por qué", no el "qué"
- Usa JSDoc para funciones públicas
- TODOs en formato: // TODO(nombre): descripción

## Commits
- Prefiero commits pequeños y atómicos
- Formato: tipo(scope): descripción
- Tipos: feat, fix, refactor, test, docs, chore
